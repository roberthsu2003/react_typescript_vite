# 09 - React 單元（進階）

## 說明

本單元與 [08 - TypeScript 語法](../08-TypeScript語法/主題.md) 並列於 [README 深度學習表格](../README.md)，內容比主線的 **[05 - React 核心概念](../05-React核心概念/主題.md)** **更深入**：在已會 JSX、Props、State、`useEffect`、事件的基礎上，進一步探討 **ref、效能 Hook、Context、自訂 Hook、表單與組合** 等主題。

### 學習順序

1. 先完成 **[05 - React 核心概念](../05-React核心概念/主題.md)** 的範例 0～5。  
2. 再依序完成本單元 **範例 0～7**（建議搭配 [08 - TypeScript 語法](../08-TypeScript語法/主題.md) 強化型別）。

### 學習目標

- 使用 `useRef` 操作 DOM 與保存可變值  
- 理解 `useMemo` / `useCallback` 與依賴陣列  
- 以 Context 跨層傳遞資料  
- 撰寫可重複使用的自訂 Hook  
- 處理較複雜的受控表單與驗證  
- 運用 `children` 與組合模式整理 UI  
- 認識 `memo`、列表 `key` 與常見效能觀念
