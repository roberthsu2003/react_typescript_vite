# 02 - Vite 入門

## 主題說明

本單元學習 Vite 是什麼、為什麼使用它，以及開發與建置的流程。Vite 以 **ES Module** 為核心（開發時原生 ESM、`import` 與 React 元件寫法皆同前置單元慣例）。

### 學習目標

- 了解 Vite 的用途與優勢
- 認識 vite.config.ts 的設定
- 理解 dev、build、preview 的差異
- 使用 **`npm create vite@latest`** 建立 **TypeScript** 或 **TypeScript + React** 專案
