# 範例 0：建立專案

## 目標

一步一步建立 `examples` 專案，作為後續範例 1～5 的基礎。完成本範例後，你將有一個可執行的專案框架，並在範例 1～5 中逐步完成各功能。

---

## 步驟 1：建立 Vite 專案

在 `05-React核心概念` 資料夾中開啟終端機，執行：

```bash
cd 05-React核心概念
npm create vite@latest examples -- --template react-ts
```

> **若已存在 examples 資料夾**：可先重新命名為 `examples-backup`，或刪除後再執行上述指令。

---

## 步驟 2：進入專案並安裝依賴

```bash
cd examples
npm install
```

---

## （建議）認識 `react-ts` 模板的內建檔案

安裝依賴後、開始改程式前，可先閱讀 **[補充：React + TypeScript + Vite 專案結構說明](範例0-專案結構說明-react-ts.md)**，了解 `index.html`、`main.tsx`、`App.tsx`、樣式與設定檔的用途，之後修改 `App.tsx` 或新增 `src/examples/` 會較有方向。

## 步驟 6：修改 App.tsx

將 `src/App.tsx` 的內容**全部替換**為：

```tsx
import { useState } from 'react'
import Example1JSX from './examples/Example1JSX'
import Example2Props from './examples/Example2Props'
import Example3State from './examples/Example3State'
import Example4UseEffect from './examples/Example4UseEffect'
import Example5Events from './examples/Example5Events'
import './App.css'

const EXAMPLES = [
  { id: 1, name: 'JSX 基礎', component: Example1JSX },
  { id: 2, name: 'Props', component: Example2Props },
  { id: 3, name: 'State', component: Example3State },
  { id: 4, name: 'useEffect', component: Example4UseEffect },
  { id: 5, name: '事件處理', component: Example5Events },
] as const

function App() {
  const [current, setCurrent] = useState(1)
  const CurrentExample = EXAMPLES.find((e) => e.id === current)?.component ?? Example1JSX

  return (
    <div className="app">
      <header className="app-header">
        <h1>React 核心概念 - 互動範例</h1>
        <p>選擇範例，透過手動操作理解每個概念</p>
      </header>

      <nav className="example-nav">
        {EXAMPLES.map(({ id, name }) => (
          <button
            key={id}
            className={current === id ? 'active' : ''}
            onClick={() => setCurrent(id)}
          >
            {id}. {name}
          </button>
        ))}
      </nav>

      <main className="example-content">
        <CurrentExample />
      </main>
    </div>
  )
}

export default App
```
