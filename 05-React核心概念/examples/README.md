# React 核心概念 - 互動範例

本專案包含 5 個可執行的 React 範例，學生可透過**手動操作**理解每個核心概念。

**學習方式**：依序完成 [範例 0](../範例0-建立專案.md)～[範例 5](../範例5-事件處理.md)，即可從零建立此專案。  
若需對照 **`react-ts` 模板**內建檔案說明，見 [專案結構補充](../範例0-專案結構說明-react-ts.md)。

## 啟動方式

```bash
npm install
npm run dev
```

瀏覽器開啟後，使用上方按鈕切換不同範例並進行操作。

## 範例列表

| 範例 | 概念 | 操作重點 |
|------|------|----------|
| 1. JSX 基礎 | `{}` 插入表達式 | 觀察變數、運算、函式呼叫的顯示 |
| 2. Props | 父傳子參數 | 觀察同元件不同參數的差異 |
| 3. State | useState | 點擊按鈕觀察數字即時更新 |
| 4. useEffect | 副作用 | 觀察計時器每秒增加 |
| 5. 事件處理 | onClick、onChange | 輸入文字、點擊按鈕 |
