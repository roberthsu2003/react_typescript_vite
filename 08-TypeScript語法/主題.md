# 08 - TypeScript 語法

## 主題說明

本單元專門講解 **TypeScript 語法**：變數與型別註解、函式、介面、泛型、聯合型別、工具型別等，讓你在閱讀與撰寫 React / Vite 專案時能對型別系統有完整概念。

對象為已讀過 [03 - TypeScript 配置](../03-TypeScript配置/主題.md)（tsconfig、`.ts` / `.tsx`）的初學者；本單元**不**重複專案設定，聚焦在**程式語法**。

---

## ⚡ 快速起步（5 分鐘）

> 第一次來？先按這個流程跑起來：

```bash
# 1. 進入指令列練習資料夾
cd 08-TypeScript語法/tsx-cli

# 2. 安裝依賴（只需一次）
npm install

# 3. 執行範例 1 示範程式，立刻看到輸出！
npm run 01
```

看到終端機有輸出後，打開 `src/01-basics.ts`，仿照示範程式自己練習。  
完成後依序執行 `npm run 02`～`npm run 06`。

## 學習目標

- 熟練使用型別註解與基本型別
- 為函式參數與回傳值標註型別
- 使用 `interface` 與 `type` 描述物件與結構
- 認識陣列、泛型與常見集合型別
- 理解聯合型別、型別窄化與實務常用寫法
- 使用工具型別（`Partial`、`Pick`、`Omit`）與型別斷言（`as`、`as const`）

---

## 學習流程

每個範例的建議學習方式：

1. **閱讀講義 `.md`** — 了解概念與程式碼範例
2. **執行 tsx-cli 示範** — `npm run 0N` 看到預期輸出，確認理解
3. **清空並自行重寫** — 對照講義步驟，從頭練習
4. **開啟 examples 網頁版**（選用）— `npm run dev` 在瀏覽器看輸出
